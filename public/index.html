Fonctionnalités principales

Captures d'écran automatiques

Enregistrez automatiquement des captures d'écran à chaque étape de vos tests pour un débogage visuel.

Intégration CI/CD

S'intègre parfaitement avec GitHub Actions et d'autres outils de CI/CD pour une automatisation complète.

Rapports détaillés

Générez automatiquement des rapports détaillés avec des captures d'écran pour une analyse approfondie.

Documentation rapide

Installation

$ git clone https://github.com/username/cypress-demo.git
$ cd cypress-demo
$ npm install

Exécution locale

$ npm start
$ npm run cypress:open

Configuration GitHub Actions

name: Cypress Screenshots CI

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]
